Protea Flowers

  • HTML
  • CSS
  • JavaScript

Esse projeto foi desenvolvido com objetivo de ser um e-commerce para uma floricultura especializada em flores tropicais. Foi utilizado a ferramenta Figma para criar o design e as telas do site. Funcionalidades de exibição, compra, pagamento e envio de produtos foram implementadas, além de criar uma área para o próprio vendedor gerenciar seus produtos, vendas, estoque e pedidos. Tudo foi pensado para criar uma experiência de usuário agradável, acessível e funcional para os visitantes.



Road Tech

  • HTML
  • CSS
  • JavaScript

O Road Tech foi desenvolvido visando ajudar jovens talentosos a encontrarem seu caminho no mundo da tecnologia. Durante esse projeto, tive a oportunidade de visitar escolas e compartilhar um pouco do meu conhecimento sobre as opções de carreira que as áreas da tecnologia oferecem. Acredito que o conhecimento e a orientação são as chaves para abrir as portas da oportunidade no campo da tecnologia. Além do HTML, CSS e JS, foi usado a biblioteca Slick para o gerênciamento do carrossel no header da página.
